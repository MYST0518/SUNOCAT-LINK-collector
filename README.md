# 🐱 SUNOCAT Link Collector

SUNOの曲リンクを簡単に収集して、[SUNOCAT Player](https://suno-playlist.vercel.app/)に一括で貼り付けられるChrome拡張機能です。

![SUNOCAT Link Collector](icon.png)

## ✨ 機能

- ✅ **ワンクリックでリンク収集**: SUNOの曲ページを開いて「➕ストックに追加」ボタンを押すだけ
- ✅ **自動貼り付け**: SUNOCAT Playerを開くと自動でリンクが入力される
- ✅ **重複チェック**: 同じリンクは自動で除外
- ✅ **シンプルなUI**: 直感的に使える操作画面
- ✅ **軽量**: わずか数MBのコンパクトな拡張機能

## 🚀 インストール方法

### 📦 手順

1. **このリポジトリをダウンロード**
   - 緑色の「Code」ボタン → 「Download ZIP」をクリック
   - ZIPファイルを解凍

2. **Chromeで拡張機能を読み込む**
   - Chrome を開く
   - アドレスバーに `chrome://extensions/` と入力
   - 右上の「**デベロッパーモード**」をONにする
   - 「**パッケージ化されていない拡張機能を読み込む**」をクリック
   - 解凍したフォルダを選択

3. **完了！**
   - ツールバーに🐱アイコンが表示されます

## 🎯 使い方

### 1. SUNOで曲リンクを収集

1. **SUNO** ([suno.com](https://suno.com/)) で気になる曲のページを開く
2. 拡張機能アイコン🐱をクリック
3. 「**➕ ストックに追加**」ボタンをクリック
4. 複数の曲を繰り返し追加できます

### 2. SUNOCAT Playerで再生

1. 「**📤 SUNOCAT PLAYERで開く**」ボタンをクリック
2. 自動的に[SUNOCAT Player](https://suno-playlist.vercel.app/)が開き、リンクが入力されます
3. 「**プレイリスト作成**」ボタンを押すだけ！

## 🛠️ 技術情報

- **Manifest Version**: 3
- **必要な権限**:
  - `storage`: リンクをローカルに保存
  - `activeTab`: 現在開いているタブのURLを取得
  - `scripting`: SUNOCAT Playerでの自動貼り付け

## 🐛 トラブルシューティング

### 拡張機能が読み込めない
- Chromeのバージョンが最新か確認してください
- デベロッパーモードがONになっているか確認してください

### リンクが追加されない
- SUNOの曲ページ（`suno.com/song/...`）を開いているか確認してください

### 自動貼り付けが動作しない
- ポップアップブロックが有効になっていないか確認してください
- 拡張機能を再読み込みしてみてください
